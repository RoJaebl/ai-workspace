## MOC 계층 심볼

| 심볼 | 이름 | Level | 사용처 |
|------|------|-------|--------|
| `🏛` | Area | 1 | 대분류 영역 (철학, 기술, 생활 등) |
| `📚` | Topic MOC | 2 | 주제별 지식 지도 |
| `📔` | Category MOC | 3 | 세부 카테고리 분류 |

## 시계열 심볼

| 심볼 | 이름 | 사용처 |
|------|------|--------|
| `🌅` | Year | 연도별 묶음 (Level 4) |
| `📅` | Month | 월별 기록 (Level 5) |
| `🖋` | Entry | 일일 기록, 개별 엔트리 (Level 6+) |

**흐름**: `🌅(연)` → `📅(월)` → `🖋(일)`

## 특수 성격 심볼

| 심볼 | 이름 | 사용처 |
|------|------|--------|
| `🪷` | Vision | 비전, 핵심 가치, 만다라트 |
| `🏠` | Home | 볼트 최상위 인덱스, 대문 |
| `📘` | Guide | 운영 가이드, 매뉴얼, 규칙 |
| `🧩` | Module | 학습 자료, 코드 조각, 컴포넌트 |
| `🎓` | Learning | 학습 노트, 강의 정리 |
| `📄` | Document | 일반 문서, 단순 노트 |

## 선택 플로우차트

```
1. MOC 계층 구조인가?
   ├─ Level 1 (대분류) → 🏛
   ├─ Level 2 (주제) → 📚
   └─ Level 3 (카테고리) → 📔

2. 시계열 데이터인가?
   ├─ 연도별 → 🌅
   ├─ 월별 → 📅
   └─ 일별/개별 → 🖋

3. 특수 성격 파일인가?
   ├─ 비전/가치 → 🪷
   ├─ 가이드/규칙 → 📘
   ├─ 학습 자료 → 🎓
   ├─ 모듈/컴포넌트 → 🧩
   └─ 홈/대문 → 🏠

4. 그 외 → 📄 (일반 문서)
```
