## 기본 구조

```
{대분류}.{주제}{알파벳}.{숫자}{알파벳}{숫자}...
    L1      L2    L3      L4    L5    L6
```

## 레벨별 변환 규칙

| 현재 → 다음 | 변환 규칙 | 예시 |
|-------------|-----------|------|
| L2 → L3 | 알파벳 추가 | `1.1` → `1.1a` |
| L3 → L4 | `.` + 숫자 | `1.1a` → `1.1a.1` |
| L4 → L5 | 알파벳 추가 | `1.1a.1` → `1.1a.1a` |
| L5 → L6 | 숫자 추가 | `1.1a.1a` → `1.1a.1a1` |

**핵심**: Level 4 이후는 숫자↔알파벳 교대, 점(`.`) 없음

## 예시 트리

```
🏛 1 철학
├── 📚 1.1 사고
│   ├── 📔 1.1a 회고록
│   │   ├── 🌅 1.1a.1 2025
│   │   │   ├── 📅 1.1a.1a 10
│   │   │   │   ├── 🖋 1.1a.1a1 2025-10-15
│   │   │   │   └── 🖋 1.1a.1a2 2025-10-16
│   │   │   └── 📅 1.1a.1b 11
│   │   └── 🌅 1.1a.2 2026
│   └── 📔 1.1b 에세이
└── 📚 1.2 윤리
    └── 📔 1.2a 도덕철학
```

## Back Link 규칙

| 현재 Level | Back Link 대상 |
|------------|----------------|
| Level 3 | Level 2 (Topic MOC) |
| Level 4+ | 바로 위 상위 노트 |

### 예시

```yaml
# 📔 1.1a 회고록.md
back link: "[[📚 1.1 사고]]"

# 🌅 1.1a.1 2025.md
back link: "[[📔 1.1a 회고록]]"

# 📅 1.1a.1a 10.md
back link: "[[🌅 1.1a.1 2025]]"

# 🖋 1.1a.1a1 2025-10-15.md
back link: "[[📅 1.1a.1a 10]]"
```

## 원초적 파일 (Atomic Files)

독립 완결성을 가진 파일은 MOC 인덱싱에서 제외:
- `📘 가이드.md`
- `🧩 모듈.md`
- `🪷 비전.md`

제목 심볼만으로 성격 식별.
